# Firebase App Config Files

Place the Firebase mobile app config files here after downloading them from the PulseSoc Firebase project:

- `google-services.json`
- `GoogleService-Info.plist`

These files are app configuration files, not Firebase service account private keys. Review them before committing. Never place service account JSON, private keys, APNs private keys, or Google Play service account credentials in this folder.

Expo config references:

- Android: `./credentials/firebase/google-services.json`
- iOS: `./credentials/firebase/GoogleService-Info.plist`
